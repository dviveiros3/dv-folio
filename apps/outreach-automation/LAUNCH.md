## Quick Start (5 Minutes)

### 1. Install Dependencies
```bash
cd apps/outreach-automation
npm install
```

### 2. Set Environment Variables
Create `.env` file:
```bash
RESEND_API_KEY=re_xxxxx  # Get from resend.com
EMAIL_FROM=Daniel <your-email@domain.com>
```

### 3. Run the System
```bash
# One-time test run
node automated-outreach.js

# Start daily scheduler
node scheduler.js
```

### 4. Monitor Dashboard
Open the React dashboard to see:
- Leads generated
- Emails sent
- Responses received
- Calls booked
- Revenue closed
