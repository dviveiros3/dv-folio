# Email Sage

AI-Powered Cold Email Automation That Actually Gets Replies

## What It Does
- Researches each prospect in real-time
- Writes personalized cold emails automatically
- Tests A/B variants continuously
- Optimizes send times per recipient

## Quick Start
1. npm install
2. cp .env.example .env
3. Add your API keys
4. npm run dev

## Tech Stack
- Next.js + TypeScript + Tailwind CSS
- PostgreSQL + Prisma ORM
- OpenAI for email generation
- SendGrid/SES for delivery
- Stripe for billing
