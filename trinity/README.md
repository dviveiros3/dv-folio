# Trinity Project

> A complete product suite of tested AI workflows, playbooks, applications, and ready-to-deploy skills.

## Core Principle
Stop building demos. Start shipping products.

## Structure
- `/apps` — Complete, deployable AI-powered applications
- `/workflows` — Reusable, tested AI automation workflows
- `/playbooks` — Step-by-step guides for solving real business problems
- `/skills` — Modular, reusable AI skills (tools, integrations, agents)
- `/marketing` — Sales copy, landing pages, pricing, go-to-market strategy
- `/research` — Market validation, competitor analysis, pricing experiments

## Rules
1. Only work inside this `trinity/` folder
2. Every deliverable must be tested and sellable
3. Build what people will pay for, not what sounds cool
4. Document everything. Confusion doesn't sell.
