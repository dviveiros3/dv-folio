# Competitor Analysis Methodology

## Research Template

### 1. Identify Direct Competitors
Search terms: [Product category] + [target segment]
- G2, Capterra, Product Hunt listings
- Google Search first 3 pages
- Social media mentions
- Industry reports (Gartner, Forrester)

### 2. Feature Matrix
| Feature | Us | Competitor A | Competitor B | Competitor C |
|---------|-----|-------------|-------------|-------------|
| Core Feature 1 | | | | |
| Core Feature 2 | | | | |
| Integrations | | | | |
| AI Capabilities | | | | |
| Pricing | | | | |
| Support | | | | |

### 3. Positioning Map
Create 2x2 matrix:
- X-axis: Low price → High price
- Y-axis: Low AI capability → High AI capability
- Plot competitors and Trinity

### 4. SWOT Per Competitor
**Strengths**: What do they do well?
**Weaknesses**: Where do they fail?
**Opportunities**: What are they ignoring?
**Threats**: What would they do if they saw us?

### 5. Pricing Intel
- Public pricing on website?
- Freemium available?
- Enterprise pricing?
- Hidden costs (setup, overage, support)?
- Discounting behavior?

### 6. Channel Strategy
- How do they acquire customers? (SEO, ads, content, sales)
- Partnership strategy?
- Community presence?
- How do they convert? (Free trial, demo, sales call)

### 7. Content Strategy
- Blog quality and frequency?
- Thought leadership presence?
- Case studies and social proof?
- Free tools or resources?

### 8. Customer Sentiment
- G2/Capterra reviews (read negative ones first)
- Reddit/Quora mentions
- Social media comments
- Support forum complaints

### 9. Update Cadence
- Review top 5 competitors monthly
- Update full analysis quarterly
- Track pricing changes immediately
