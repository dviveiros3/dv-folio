# Sellable Product Criteria Checklist

## Must-Have (Non-Negotiable)
- [ ] Solves a specific, high-pain problem
- [ ] Has a clear, specific target customer
- [ ] Can be demonstrated with before/after
- [ ] Has a working MVP or prototype
- [ ] Is documented and reproducible

## Business Viability
- [ ] Total addressable market > $1M
- [ ] Customer acquisition cost < 33% of lifetime value
- [ ] Unit economics are profitable at scale
- [ ] Can be priced between $49-$999/month
- [ ] Has at least 3 potential customer segments

## Technical Quality
- [ ] Handles edge cases without breaking
- [ ] Has error handling and recovery
- [ ] Is secure (no exposed keys, no PII leaks)
- [ ] Has automated testing coverage > 80%
- [ ] Works reliably without constant babysitting
- [ ] Has usage analytics and monitoring
- [ ] Can be deployed in < 30 minutes
- [ ] Has rollback capability

## Market Readiness
- [ ] Has a clear value proposition statement
- [ ] Has a pricing page with clear tiers
- [ ] Has a demo or sample output
- [ ] Has customer testimonials or case studies
- [ ] Has a refund policy and terms of service
- [ ] Sales page conversion rate > 2%
- [ ] Free trial-to-paid conversion > 15%

## Documentation
- [ ] README with setup instructions
- [ ] API documentation (if applicable)
- [ ] User guide or tutorial
- [ ] FAQ for common issues
- [ ] Changelog for updates
- [ ] Roadmap for future versions

## Support & Maintenance
- [ ] Monitored uptime and alerts
- [ ] Response time < 24 hours for support
- [ ] Regular update cadence (monthly minimum)
- [ ] Deprecation and migration plan if needed
