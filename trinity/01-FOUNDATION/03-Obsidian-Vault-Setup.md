# Trinity Obsidian Vault

## Structure
- **00-Inbox** — Raw ideas, quick notes, unprocessed
- **01-Products** — Active product development notes
- **02-Research** — Market analysis, competitor intel
- **03-Customers** — Avatar details, interview notes
- **04-Playbooks** — Internal SOPs and processes
- **05-Marketing** — Copy, content, campaign notes
- **06-Analytics** — Metrics, experiments, results
- **07-Archive** — Completed projects, old versions

## Naming Conventions
- Use Title Case for page names
- Prefix dates: 2026-05-15 Weekly Review
- Use atomic notes: One idea per note
- Link liberally using [[WikiLinks]]
- Tag system: #product #research #marketing #urgent
