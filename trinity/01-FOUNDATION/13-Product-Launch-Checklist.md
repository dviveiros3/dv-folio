# Product Launch Checklist Template

## 4 Weeks Before
- [ ] Finalize product positioning and messaging
- [ ] Complete all testing and QA
- [ ] Prepare pricing and packaging
- [ ] Create landing page and sales copy
- [ ] Set up analytics and tracking
- [ ] Prepare demo video or walkthrough
- [ ] Draft press release and PR strategy
- [ ] Identify launch partners and affiliates
- [ ] Create email announcement sequence

## 2 Weeks Before
- [ ] Beta testing feedback incorporated
- [ ] Sales team trained on product
- [ ] Support documentation complete
- [ ] Onboarding flow tested
- [ ] Payment processing tested end-to-end
- [ ] Refund process documented
- [ ] Legal terms finalized (ToS, Privacy, Refund)
- [ ] Social media content scheduled
- [ ] Influencer outreach initiated

## 1 Week Before
- [ ] Soft launch to waitlist and early customers
- [ ] Monitor for critical bugs
- [ ] Load test infrastructure
- [ ] Prepare FAQ for common questions
- [ ] Final review of all copy and assets
- [ ] Schedule launch day outreach
- [ ] Brief team on launch day procedures
- [ ] Set up war room (Slack channel or physical)

## Launch Day
- [ ] Official announcement across all channels
- [ ] Monitor analytics in real-time
- [ ] Respond to comments and questions immediately
- [ ] Track conversion rate
- [ ] Address any technical issues immediately
- [ ] Send thank you to early adopters
- [ ] Capture testimonials from first customers
- [ ] Post-launch retrospective scheduled

## Post-Launch (Week 1)
- [ ] Review all metrics vs targets
- [ ] Customer interviews for feedback
- [ ] Iterate on messaging based on response
- [ ] Plan first update or patch
- [ ] Analyze top acquisition channels
- [ ] Optimize based on funnel data
- [ ] Update roadmap based on early feedback
