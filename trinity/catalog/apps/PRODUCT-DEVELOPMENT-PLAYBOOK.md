# Product Development Playbook

## Phase 1: Ideation
1. Identify painful problem
2. Validate market exists (>10 people would pay)
3. Check competition (is someone doing this already?)
4. Define MVP scope (what can we ship in 2 weeks?)
5. Set success metrics (how do we know it works?)

## Phase 2: Prototype
1. Build the simplest version that demonstrates value
2. Test with 3-5 target users
3. Collect qualitative feedback
4. Iterate based on feedback
5. Test again

## Phase 3: MVP
1. Build core features only
2. Polish UX enough for paid users
3. Set up basic analytics
4. Create onboarding flow
5. Start charging (even if just $5)

## Phase 4: Validation
1. Track retention (do users come back?)
2. Track NPS (do users recommend?)
3. Track conversion (do free users pay?)
4. Interview customers weekly
5. Measure against success criteria

## Phase 5: Growth
1. Optimize conversion funnel
2. Add features that paying users request
3. Improve onboarding and activation
4. Expand to new customer segments
5. Scale marketing channels

## Phase 6: Maturity
1. Operationalize support
2. Build team and processes
3. Consider strategic partnerships
4. Explore new markets
5. Plan next product or pivot
