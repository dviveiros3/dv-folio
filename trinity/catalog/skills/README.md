# Trinity Skills

Modular, reusable AI skills (tools, integrations, and agent capabilities) that can be packaged, licensed, and sold.

## Skill Criteria
- [ ] Solves a well-defined sub-problem
- [ ] Can be used standalone or composed into larger systems
- [ ] Well-documented API or CLI
- [ ] Includes tests and benchmarks
- [ ] Ready for marketplace listing
