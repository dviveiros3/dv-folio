# Playbook: Launch an AI-Powered Cold Email Tool

## Overview
Step-by-step guide to launching an AI cold email tool from zero to first paying customer.

## Phase 1: Validate (Days 1-3)
- Interview 10 SDRs about cold email pain
- Quantify current workflow and costs
- Document frustrations and desired features

## Phase 2: Define MVP (Days 4-7)
- List all features, prioritize by impact/effort
- Define core loop: research, generate, send, analyze
- Create wireframes and user stories

## Phase 3: Build (Weeks 2-3)
- Set up Next.js + DB + AI provider
- Implement LinkedIn research, email generation, sequences
- Build dashboard, analytics, Stripe billing
- Set up CI/CD

## Phase 4: Beta (Week 4)
- Recruit 5 beta testers from interviews
- Daily feedback via Slack or Loom
- Iterate on biggest pain points

## Phase 5: Launch (Week 5)
- Product Hunt, Hacker News, Reddit
- Email blast to network and beta list
- Social media with demo video
- Influencer outreach
- Monitor and respond

## Phase 6: Iterate and Scale
- Track activation, retention, NPS
- Build features paying users request
- Optimize onboarding and conversion
- Expand to adjacent use cases
