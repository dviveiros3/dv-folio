## Development

### Local Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Run linting checks:
   ```bash
   npm run lint
   ```
4. Build for production:
   ```bash
   npm run build
   ```

### Project Structure

- `app/`: Next.js App Router pages and layout.
- `components/`: Product-ready UI components.
- `data/`: Content and experience data.
- `public/`: Production assets.
- `scratch/trinity/`: Raw research, experiments, and placeholder assets (not used in production).
