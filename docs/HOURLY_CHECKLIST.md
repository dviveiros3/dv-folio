## BLOCKER CHECK (If stuck, answer these)

**Not sending emails?**
- [ ] I'm afraid of rejection
- [ ] I don't have 50 companies
- [ ] I don't know what to say
- [ ] I'm waiting for "the right time"

**SOLUTION:** Just send. One email. Now. Then another. Imperfect action > perfect inaction.

**Not getting responses?**
- [ ] My subject line sucks
- [ ] I'm not personalizing
- [ ] I'm not offering value
- [ ] I'm giving up too soon

**SOLUTION:** Change subject to "Found $47k in waste". Add their company name in first line. Send 50, not 5.

**Not booking calls?**
- [ ] I'm not asking for the call
- [ ] I'm not creating urgency
- [ ] I'm not following up
- [ ] I sound desperate

**SOLUTION:** "Worth 15 mins to see where you're wasting 30%? I have 3 audit spots this week."

**Not closing on calls?**
- [ ] I'm talking too much
- [ ] I'm not asking pain questions
- [ ] I'm not offering the guarantee
- [ ] I'm not asking for the sale

**SOLUTION:** Ask "What's your current CAC?" then "What would it mean if we cut that by 30%?" then "Should we start Monday?"
